smProbLog is a PLP framework based on ProbLog2, whose semantics supports probabilistic logic programs where a choice of probabilistic facts does not yield a unique interpretation of the logical atoms in the program.
  • Traditional PLP frameworks assume that each possible world yields exactly one valid interpretation of the logical atoms.
  • This means that there is no room for non-determinism other than probabilistic facts
  • This is not the case when arguments form cyclic dependencies involving attacks.
  • smProbLog uses stable model semantics in each possible world.
